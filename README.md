# Diablo 2 Item Store

A (fake) Diablo 2 item store to practice with Laravel 8 & Vue 3

# Installation

### 1.) Clone the Repo:

```bash
git clone https://github.com/0xRenegade/D2-Laravel-Commerce-App.git
```

### 2.) Install composer dependencies
```bash
composer install
```

### 3.) Edit Environment Variables 

```bash
cp .env.example .env
```

### 4.) Run project with [Docker & Laravel Sail](https://docs.docker.com/get-docker/)

```bash
sail up
```

### 5.) Compile frontend assets

```bash
sail npm run watch
```
